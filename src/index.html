Simplicity in mind

OpenLocalization has been designed to be very easy to use. No complicated settings or uncomprehensible menu. The UI/UX aims to be as simple as possible without losing too many features.

Try the live demo

Flexible

Developed as an API first, it should work for any application, no matter the programming language you use. Simply drop the configuration file in the root of your project, and you're done.

Learn how it works

Run your own instance

Don't be dependent on a third-party service. Run this web-app on your servers (integration with LDAP soon). It's free, it's open-source. You can even contribute to adapt it better for your business.

Why this new translation app?

I started this project because I couldn't find any good and free translation tools for a simple usage. You may find some nice SaaS products (eg. Transifex, Crowdin), but not enough satisfying if you don't want to depend on an external service, or if you have really simple use cases.

OpenLocalization goal is to bring a powerful tool that will match any (or most of) needs, and which could be easily integrated in your development workflow.
